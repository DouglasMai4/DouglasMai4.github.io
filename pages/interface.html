Interfaces Gráficas

GNOME

Vamos instalar o gerenciador de login: pacman -S gdm
Agora vamos ativar para iniciar com o sistema: systemctl enable gdm

pacman -S gnome gnome-terminal nautilus gnome-tweaks gnome-control-center gnome-backgrounds adwaita-icon-theme

Plasma

Vamos instalar o gerenciador de login: pacman -S sddm
Agora vamos ativar para iniciar com o sistema: systemctl enable sddm

pacman -S plama konsole dolphin gwenview elisa dragonplayer ark kcalc kwrite

XFCE

Vamos instalar o gerenciador de login: pacman -S lightdm lightdm-gtk-greeter lightdm-gtk-greeter-settings
Agora vamos ativar para iniciar com o sistema: systemctl enable lightdm

pacman -S xfce4 xfce4-goodies xfce4-terminal xfce4-whiskermenu-plugin
Não esqueça de instalar um navegador como o firefox por exemplo.
