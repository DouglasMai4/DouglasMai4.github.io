Meta Tag

Garante a codificação correta --> <meta charset="UTF-8" />

Descrição do conteudo do site --> <meta name="description" content="Posso usar essa meta tag para descrever informações do meu site">

Palavras chave --> <meta name="keywords" content"html5, html, aula, exemplos, Canoinhas, SC">

Autor --> <meta name="autor" content="Douglas Maia">

Para orientar os robôs de busca (fallow e nofallow) --> <meta name="robots" content="index, follow">

Atualiza a página a cada 3 segundos --> <meta http-equiv="refresh" content="3">

Sempre usar ícones com extensão .ico --> <link rel="icon" href="img/html5.ico">

Area em que o conteudo pode ser acessado (Importante para dispositivos móveis) --> <meta name="viewport" content="width=device-width, initial-scale=1.0" />
