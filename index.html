A origem do Arch Linux

Foi desenvolvido originalmente pelo Canadense Judd Vinet, seu código sempre foi focado no minimalisno e simplicidade, esperando que o usuário seja quem vai trazer a coesão esperada para o sistema, ele utiliza do modelo Rolling Release de lançamento, isso quer dizer que não existe "Versões" do Arch como temos em outras distribuições, a ideia é que o usuário instale uma vez e vá atualizando o sistema constantemente.
